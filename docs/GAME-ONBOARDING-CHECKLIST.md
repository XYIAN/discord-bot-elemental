# Game Onboarding Checklist

Use this checklist to adapt the template to a new game/community.

## Branding and naming

- [ ] Final game name decided (Legend of Elements)
- [ ] Main server/community name finalized (keep configurable until decided)
- [ ] Clan/community brand decided (Tempest)
- [ ] Persona style guide drafted
- [ ] Channel naming convention chosen (train-case)

## Discord setup

- [ ] New Discord bot application created
- [ ] Bot invited with correct permissions
- [ ] Required channels created
- [ ] Category and permission model finalized
- [ ] Role hierarchy created
- [ ] Tempest private clan section created
- [ ] Tempest clan roles created (`XY Tempest Verified`, `XY Tempest Officer`)

## Bot config

- [ ] Role names mapped in bot config
- [ ] Clan terminology set (replace guild wording where needed)
- [ ] Activity channels mapped
- [ ] Command permissions mapped
- [ ] Welcome copy updated for new game

## Knowledge setup

- [ ] `knowledge.json` category schema defined
- [ ] Starter facts seeded
- [ ] Suggestion moderation flow tested

## Deployment

- [ ] Railway project created
- [ ] Env vars set in Railway
- [ ] Healthcheck verified
- [ ] First deploy successful
- [ ] Debug + changelog posts confirmed
